# خارطة طريق دليل السودان الرقمي

هذه الوثيقة تمثل النسخة التنفيذية الأولى للمشروع داخل الكود، حتى يكون المشروع قابلاً للبناء التدريجي والتسليم لاحقاً لجهة رسمية.

## الهدف

إنشاء منصة وطنية توثق:

- الولايات السودانية
- المحليات
- التاريخ والهوية الثقافية
- فرص الاستثمار
- الشخصيات البارزة
- دليل الخدمات والأرقام المهمة
- مساهمات المجتمع وسفراء الولايات

## النواة التقنية الحالية

تم تأسيس الطبقات الأساسية التالية:

- `states`: تعريف الولاية ومعلوماتها العامة
- `localities`: المحليات التابعة لكل ولاية
- `categories`: تصنيفات المحتوى
- `entries`: المحتوى الرسمي المعتمد
- `services`: دليل الخدمات والهواتف
- `contributions`: المساهمات المفتوحة قبل المراجعة
- `users`: ربط المستخدم بالولاية والدور والحالة

## دورة المحتوى

1. يرسل المواطن أو السفير مساهمة جديدة.
2. تحفظ المساهمة بحالة `pending`.
3. يراجعها الآدمن أو المراجع.
4. بعد الاعتماد يتم تحويلها إلى `entry` رسمي.
5. يبقى أثر المصدر محفوظاً داخل `meta`.

## الأدوار المقترحة

- `super_admin`: إدارة المنصة كاملة
- `moderator`: مراجعة واعتماد المحتوى
- `ambassador`: إدارة محتوى ولاية محددة
- `citizen`: مساهم عام أو مستخدم عادي

## المراحل التالية

### المرحلة 1

- بناء Filament Resources للولايات والمحليات والتصنيفات والمحتوى

- إعداد Seeder للولايات السودانية

### المرحلة 2
